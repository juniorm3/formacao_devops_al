Como fazemos para o Git passar a enxergar determinada pasta como um repositório e a observar as mudanças em seus arquivos?
- O git init inicializa um repositório no diretório em que o comando for executado. A partir deste comando, o Git poderá gerenciar as 
modificações realizadas nos arquivos.
